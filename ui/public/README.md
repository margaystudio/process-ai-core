# Public Assets

Esta carpeta contiene archivos estáticos que se sirven directamente.

## Logo del Margay

Para agregar el logo del margay:

1. Coloca tu imagen en esta carpeta con el nombre `margay-logo.png`
2. Formatos soportados: PNG, SVG, JPG
3. Tamaño recomendado: 40x40px o mayor (se escalará automáticamente)
4. El header mostrará automáticamente la imagen

Si no existe la imagen, se mostrará un SVG de ejemplo de un margay.
